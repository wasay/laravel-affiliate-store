## Laravel-affiliate-store

#### This is a starting boilerplate for an affiliate store build with Laravel.

* Design ready
* Dashboard admin panel
* Google analytics ready


If you want to clone -

* clone this repo
* composer install
* php artisan key:generate
* cp env.example env
* modify env file
* php artisan db:seed
